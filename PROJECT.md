# Cannabis Strain Database

## Goal
Build a structured, scalable cannabis strain database.

## Requirements
- Consistent JSON structure
- Extendable dataset
- No breaking changes

## Fields
- name
- breeder
- thc
- cbd
- terpenes
- genetics
- effects
- flavors
- flowering_time_weeks
- yield
- difficulty

## Rules
- Never modify existing structure
- Only extend data
- Keep values realistic
